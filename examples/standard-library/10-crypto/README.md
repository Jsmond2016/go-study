# 加密示例

本示例展示了 Go 语言 crypto 包的哈希功能，包括 MD5、SHA1 和 SHA256。

## 运行示例

```bash
go run main.go
```

## 知识点

### 哈希算法

- `md5.Sum()`：计算 MD5 哈希值
- `sha1.Sum()`：计算 SHA1 哈希值
- `sha256.Sum256()`：计算 SHA256 哈希值

### Hash 接口

- `hash.Hash` 接口提供了统一的哈希操作
- `New()`：创建哈希实例
- `Write()`：写入数据
- `Sum()`：计算哈希值

## 注意事项

- MD5 和 SHA1 已不安全，不推荐用于安全场景
- 推荐使用 SHA256 或更安全的算法
- 对于密码存储，应使用 bcrypt 或 argon2
