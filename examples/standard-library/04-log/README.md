# 日志示例

本示例展示了 Go 语言 log 包的使用，包括基本日志输出、日志级别、自定义日志格式和文件输出等。

## 运行示例

```bash
go run main.go
```

## 知识点

### 基本日志输出

- `log.Print()`：普通日志输出
- `log.Println()`：带换行的日志输出
- `log.Printf()`：格式化日志输出

### 日志级别

- `log.Fatal()`：输出日志后调用 `os.Exit(1)`
- `log.Panic()`：输出日志后调用 `panic`

### 自定义日志

- `SetPrefix()`：设置日志前缀
- `SetFlags()`：设置日志标志
- `SetOutput()`：设置日志输出目标
