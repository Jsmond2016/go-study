# 数学运算示例

本示例展示了 Go 语言 math 包的使用，包括基本数学函数、三角函数、对数函数等。

## 运行示例

```bash
go run main.go
```

## 知识点

### 基本数学函数

- `math.Pi`：圆周率
- `math.E`：自然常数
- `math.Sqrt()`：平方根
- `math.Pow()`：幂运算

### 三角函数

- `math.Sin()`：正弦
- `math.Cos()`：余弦
- `math.Tan()`：正切
- 角度使用弧度制

### 对数函数

- `math.Log()`：自然对数
- `math.Log10()`：常用对数
- `math.Log2()`：以2为底的对数

### 取整函数

- `math.Ceil()`：向上取整
- `math.Floor()`：向下取整
- `math.Round()`：四舍五入

## 练习

1. 实现一个计算器，支持基本数学运算
2. 创建一个函数，计算圆的面积和周长
3. 实现一个简单的统计函数，计算平均值、标准差等
