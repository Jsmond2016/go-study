# 文件操作示例

本示例展示了 Go 语言的文件操作，包括文件读写、目录操作和文件信息获取等。

## 运行示例

```bash
go run main.go
```

## 知识点

### 文件读写

- `os.WriteFile()`：写入整个文件
- `os.ReadFile()`：读取整个文件
- `os.Open()`：打开文件
- `bufio.Scanner`：逐行读取

### 文件信息

- `os.Stat()`：获取文件信息
- `os.IsNotExist()`：检查文件是否存在

### 目录操作

- `os.Mkdir()`：创建目录
- `os.ReadDir()`：读取目录内容
- `os.Remove()`：删除文件或目录
