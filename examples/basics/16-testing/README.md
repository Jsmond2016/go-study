# 测试示例

本示例展示了 Go 语言的测试，包括单元测试、基准测试和示例测试。

## 运行测试

```bash
# 运行所有测试
go test

# 运行测试并查看详细信息
go test -v

# 运行测试并查看覆盖率
go test -cover

# 运行基准测试
go test -bench=.

# 运行基准测试并查看内存分配
go test -bench=. -benchmem
```

## 知识点

### 单元测试

- 测试文件以 `_test.go` 结尾
- 测试函数以 `Test` 开头
- 使用 `testing.T` 进行测试

### 表驱动测试

- 使用结构体切片定义测试用例
- 使用 `t.Run()` 运行子测试
- 便于添加新的测试用例

### 基准测试

- 函数以 `Benchmark` 开头
- 使用 `testing.B` 进行基准测试
- `b.N` 是迭代次数

### 示例测试

- 函数以 `Example` 开头
- 使用 `// Output:` 注释指定期望输出
- 可以用于文档生成
