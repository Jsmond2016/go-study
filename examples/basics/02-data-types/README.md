# 数据类型示例

本示例全面展示了 Go 语言中的各种数据类型，包括基本类型、复合类型和自定义类型。

## 运行示例

```bash
go run main.go
```

## 知识点

### 基本数据类型

1. **整型**：
   - `int8`, `int16`, `int32`, `int64`：有符号整数
   - `uint8`, `uint16`, `uint32`, `uint64`：无符号整数
   - `int`, `uint`：平台相关大小的整数
   - `uintptr`：用于存储指针的整数

2. **浮点型**：
   - `float32`：32位浮点数
   - `float64`：64位浮点数（推荐使用）

3. **复数类型**：
   - `complex64`：32位实部和虚部
   - `complex128`：64位实部和虚部

4. **布尔型**：
   - `bool`：`true` 或 `false`

5. **字符串**：
   - `string`：UTF-8 编码的字符串
   - `rune`：Unicode 字符（32位）
   - `byte`：ASCII 字符（8位）

### 复合数据类型

1. **数组**：固定长度的元素序列
2. **切片**：动态长度的元素序列
3. **映射**：键值对的集合
4. **结构体**：命名字段的聚合

### 自定义类型

1. **类型别名**：使用 `type` 关键字创建新类型
2. **自定义结构体**：用户定义的数据结构
3. **方法**：为类型定义方法
4. **接口**：定义行为契约

### 类型信息

1. **类型大小**：使用 `unsafe.Sizeof()` 获取类型大小
2. **零值**：每种类型都有默认的零值
3. **类型转换**：不同类型之间的转换规则

## 重要概念

- **零值**：变量声明但未初始化时的默认值
- **类型安全**：Go 是强类型语言，需要显式类型转换
- **内存布局**：不同类型在内存中的占用大小
- **Unicode 支持**：Go 原生支持 Unicode 字符
