# 变量与常量示例

本示例展示了 Go 语言中变量和常量的各种声明和使用方式。

## 运行示例

```bash
go run main.go
```

## 知识点

### 常量 (Constants)

1. **基本定义**：使用 `const` 关键字
2. **常量组**：使用 `const()` 定义一组常量
3. **iota 枚举**：使用 iota 创建自增常量

### 变量 (Variables)

1. **var 声明**：`var name type = value`
2. **类型推断**：`var name = value`（编译器推断类型）
3. **短声明**：`name := value`（只能在函数内使用）
4. **多变量声明**：并行声明多个变量
5. **变量交换**：Go 支持简单的变量交换

### 类型转换

- 显式类型转换：`type(value)`
- 注意不同类型之间的转换规则
- 字符串转数字需要使用 `strconv` 包

### 变量作用域

- 全局变量：在包级别声明
- 局部变量：在函数或代码块内声明
- 变量遮蔽：内部作用域可以遮蔽外部变量

### 特殊函数

- `init()` 函数：在 `main()` 函数之前执行，用于初始化操作
