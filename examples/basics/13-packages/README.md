# 包管理示例

本示例展示了 Go 语言的包管理，包括包的导入、包级别变量和函数、init 函数等。

## 运行示例

```bash
go run main.go
```

## 知识点

### 包的基础

1. **包声明**：每个 Go 文件必须以 `package` 声明开头
2. **main 包**：可执行程序必须使用 `package main`
3. **导入包**：使用 `import` 关键字导入包

### 导入方式

- **标准导入**：`import "fmt"`
- **别名导入**：`import m "math"`
- **点导入**：`import . "fmt"`（不推荐）
- **空白导入**：`import _ "database/sql/driver"`（用于初始化）

### 包级别变量和函数

- 在包级别声明的变量和函数可以在包内任何地方访问
- 首字母大写的标识符是导出的（公开的）
- 首字母小写的标识符是未导出的（私有的）

### init 函数

- `init()` 函数在 `main()` 函数之前自动执行
- 可以有多个 `init()` 函数，按顺序执行
- 用于包的初始化操作

## 练习

1. 创建一个自定义包，包含导出的函数和未导出的函数
2. 在另一个包中导入并使用这个自定义包
3. 使用 init 函数初始化包级别的配置
