# 接口示例

本示例展示了 Go 语言中接口的使用，包括接口定义、实现、多态、空接口和类型断言等。

## 运行示例

```bash
go run main.go
```

## 知识点

### 接口基础

1. **接口定义**：使用 `type InterfaceName interface` 定义接口
2. **隐式实现**：类型不需要显式声明实现接口，只要实现了接口的所有方法即可
3. **多态**：通过接口实现多态，同一个接口可以有不同的实现

### 空接口

- `interface{}` 是空接口，可以表示任何类型
- Go 1.18+ 可以使用 `any` 作为 `interface{}` 的别名

### 类型断言

- 使用 `value, ok := interfaceValue.(Type)` 进行类型断言
- 使用 `switch v := value.(type)` 进行类型开关判断

### 接口组合

- 接口可以组合，形成新的接口
- 组合接口包含所有被组合接口的方法

## 练习

1. 创建一个 `Shape` 接口，包含 `Area()` 和 `Perimeter()` 方法
2. 实现 `Circle` 和 `Rectangle` 类型，都实现 `Shape` 接口
3. 创建一个函数，接受 `Shape` 接口参数，计算并打印面积和周长
4. 使用类型断言判断接口值的具体类型
