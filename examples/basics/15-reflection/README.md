# 反射示例

本示例展示了 Go 语言的反射机制，包括类型反射、值反射、结构体反射和方法反射等。

## 运行示例

```bash
go run main.go
```

## 知识点

### 反射基础

- `reflect.TypeOf()`：获取类型信息
- `reflect.ValueOf()`：获取值信息
- `Kind()`：获取底层类型

### 修改值

- 必须使用指针才能修改值
- 使用 `Elem()` 获取指针指向的值
- 使用 `SetXXX()` 方法设置值

### 结构体反射

- `NumField()`：获取字段数量
- `Field(i)`：获取第 i 个字段
- `FieldByName()`：根据名称获取字段

### 方法反射

- `NumMethod()`：获取方法数量
- `Method(i)`：获取第 i 个方法
- `MethodByName()`：根据名称获取方法
- `Call()`：调用方法

## 注意事项

- 反射会降低性能，应谨慎使用
- 反射代码较难理解和维护
- 优先考虑使用接口和类型断言

## 练习

1. 创建一个函数，使用反射打印任意结构体的所有字段和值
2. 实现一个通用的 JSON 序列化函数，使用反射处理结构体
3. 使用反射实现一个简单的依赖注入容器
