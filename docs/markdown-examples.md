# Markdown 扩展示例

本文档演示 VitePress 提供的 Markdown 扩展功能，包括任务列表、代码高亮、自定义容器等。

## ✅ 任务列表（Task Lists）

VitePress 支持 GitHub 风格的任务列表，可以用于学习进度跟踪和待办事项。

### 基本用法

**输入：**

```markdown
- [ ] 未完成的任务
- [x] 已完成的任务
- [ ] 另一个未完成的任务
```

**输出：**

- [ ] 未完成的任务
- [x] 已完成的任务
- [ ] 另一个未完成的任务

### 嵌套任务列表

- [x] 学习 Go 语言基础
  - [x] 变量与常量
  - [x] 数据类型
  - [ ] 控制流程
  - [ ] 函数
- [ ] 学习标准库
  - [ ] fmt 包
  - [ ] time 包
  - [ ] strings 包
- [ ] 实战项目

### 在文档中使用

你可以在任何文档中使用任务列表来跟踪学习进度：

```markdown
## 📋 学习目标

- [ ] 掌握变量和常量的使用
- [ ] 理解数据类型转换
- [ ] 学会使用控制流程
- [x] 完成基础语法学习
```

## 🎨 语法高亮

VitePress 提供强大的语法高亮功能，支持行号和高亮特定行。

### 基本代码块

```go
package main

import "fmt"

func main() {
    fmt.Println("Hello, Go!")
}
```

### 行号显示

```go{1-3}
package main

import "fmt"

func main() {
    fmt.Println("Hello, Go!")
}
```

### 高亮特定行

```go{4,6}
package main

import "fmt"

func main() {
    fmt.Println("Hello, Go!")
}
```

### 行号范围

```go{2-4}
package main

import "fmt"

func main() {
    fmt.Println("Hello, Go!")
}
```

## 📦 自定义容器

VitePress 支持多种自定义容器，用于突出显示不同类型的内容。

### 信息框

::: info
这是一条信息提示。用于展示一般性的说明信息。
:::

### 提示框

::: tip
这是一个提示。用于展示有用的建议和技巧。
:::

### 警告框

::: warning
这是一个警告。用于提醒需要注意的事项。
:::

### 危险框

::: danger
这是一个危险提示。用于展示重要的警告信息。
:::

### 详情折叠框

::: details 点击查看详情
这是一个可折叠的详情框，可以隐藏详细内容。

```go
func example() {
    // 这里可以放置代码示例
}
```
:::

### 自定义标题

::: tip 学习建议
建议每天学习 1-2 小时，循序渐进，不要急于求成。
:::
